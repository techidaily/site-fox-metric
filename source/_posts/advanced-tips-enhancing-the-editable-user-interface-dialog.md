## Edit UI Level Dialog

In this dialog you must specify the corresponding UI level switches for your prerequisite.

Depending on the UI level configured for the prerequisite, these switches will be appended to the command line of the prerequisites. Advanced Installer does this by appending the \[AI\_PREREQ\_UILEVEL\_SWITCH\] property to the [command line](https://tools.techidaily.com/advancedinstaller/products/) of the prerequisite. This property will be resolved according to the UI of the package.
