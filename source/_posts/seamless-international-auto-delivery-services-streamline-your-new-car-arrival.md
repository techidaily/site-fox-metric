## Auto Import

Advanced Installer has the ability to import XML and INI files. Besides the usual extensions of ".xml" and ".ini", you can define your own, so Advanced Installer will automatically detect these files when adding them using multiple selection or when adding a folder that contains these files in the [Files and Folders](https://tools.techidaily.com/advancedinstaller/products/) page.

In this edit box, you can enter all the extensions that you wish for Advanced Installer to associate with INI files.

![Note](https://cdn.advancedinstaller.com/svg/common/IconMessageNote.svg)You can delete an extension from the lists, even from the predefined ones. All extensions must be defined with spaces between them and not containing the **dot** before the actual extension, as you can see in the dialog on the right which contains the default XML and INI extensions defined by us.
